# Codex Skill Routing Rules

Generated by `scripts/build_skill_routing.py`.

## Router Protocol

- Check this file before starting a non-trivial task.
- Load one high-confidence skill directly.
- Use complementary skills together when they cover different parts of the same request.
- Ask the user to choose when matched skills are mutually exclusive.
- Treat this file as an index. After selecting a skill, read its `SKILL.md`.

## Routes

### mineru

- Source: mcp
- Priority: high

Use when:

- User provides a PDF, Word, PowerPoint, spreadsheet, image, local file path, or document URL.
- User asks to parse, read, extract text from, convert, OCR, or summarize a file.

Ask user when:

- The document appears sensitive and would be uploaded to mineru.net.

### humanizer-zh
